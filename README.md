## Getting Started

Clone the repo:

```bash
git clone https://github.com/your-username/nuxt4-tailwind-shadcn-starter.git
cd nuxt4-tailwind-shadcn-starter
```

Install dependencies:

```bash
pnpm install
# or
npm install
# or
yarn install
```

Start the dev server:

```bash
pnpm dev
# or
npm run dev
# or
yarn dev
```
